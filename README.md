## ğŸš€ Cara Pemasangan (Langkah-demi-Langkah)

Ikuti langkah-langkah di bawah ini untuk memulai:

1.  **Clone Repository** Buka Termux atau terminal kalian, lalu jalankan:
    ```bash
    git clone https://github.com/DanxyOfficial/SCRIPT-BOT-TELERAM-VERCEL
    ```

2.  **Upload ke GitHub** Push folder hasil clone tadi ke akun GitHub pribadi kalian.

3.  **Deploy ke Vercel** - Buka [Vercel.com](https://vercel.com).
    - Hubungkan akun GitHub kalian dan pilih repositori tersebut untuk di-deploy.

4.  **Konfigurasi Environment (.env)** Masuk ke bagian **Settings > Environment Variables** di dashboard Vercel, lalu tambahkan:
    - Nama Bot
    - Token Bot Telegram (Dapatkan dari [@BotFather](https://t.me/BotFather))

5.  **Edit Webhook (bot.js)** - Cari file `api/bot.js`.
    - Temukan bagian link `vercel.app`.
    - Ganti link tersebut dengan link hasil deploy Vercel milik kalian sendiri.

6.  **Redeploy & Test** Lakukan *Redeploy* pada project Vercel. Jika sudah selesai, silakan tes bot kalian di Telegram!
